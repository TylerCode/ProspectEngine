# ProspectEngine
Visual Novel engine written completely in C

# Build command (idfk)
`g++ -std=c++17 main.c -I"include" -L"lib" -Wall -lSDL2main -lSDL2 -lSDL2_image -o test`

or 

`gcc main.c -I/usr/local/include -L/usr/local/lib -lSDL2main -lSDL2 -lSDL2_image -o test`
